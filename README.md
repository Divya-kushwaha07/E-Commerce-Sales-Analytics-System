# 🛒 E-Commerce Sales Analytics System

A complete Data Analytics and Machine Learning project designed to analyze e-commerce business performance and predict customer spending behavior.

## 📖 Overview

This project simulates a real-world e-commerce business environment by generating customer, product, order, and payment data. The data is analyzed using SQL and Python, customer spending is predicted using Machine Learning, and business insights are presented through Streamlit and Power BI dashboards.

## 🎯 Project Goals

* Analyze customer purchasing behavior
* Identify revenue trends and business insights
* Predict customer spending using Machine Learning
* Build an interactive Streamlit application
* Create a professional Power BI dashboard
